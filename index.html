Hello AWS World!

Below is a summary of what was done to bring this web server online.

  1. Created a security group, ec2-sg-1, with inbound permissions to ports 22, 80 from 0.0.0.0/0
  2. Created a key pair, ec2-key-1, for SSH access
  3. Launched an t2.micro instance, using the Amazon Linux AMI
    • Used vpc-1 with no preference on AZ
    • Enabled auto-assign Public IP
    • Left IAM role as None since we will only be using the security group created above

Used the following bootstrap script:

#!/bin/bash
yum install httpd -y
yum update -y
chkconfig httpd on
service httpd start
